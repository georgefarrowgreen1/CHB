# Photos — what to upload

NOTE: `logo.svg` (the rose-gold crown logo) is already included and used in the
site header and loading screen — leave it in place; don't overwrite it with a
cottage photo. `favicon.png` and `apple-touch-icon.png` (the browser-tab icons)
are also generated from the crown — leave those too.

Put your own JPG photos in the SAME folder as `index.html`, using these exact
filenames. The site loads them automatically. Use `.jpg` files (lowercase name,
exactly as shown). Recommended size: about 1600–1920px wide, saved at good quality
but compressed for web (aim for under ~400KB each so pages load fast).

## Notes

- If a file is missing, that image area will just appear blank — the site still works.
- The filenames are case-sensitive on IONOS's servers: `Hero.JPG` will NOT match
  `hero.jpg`. Use lowercase, exactly as listed.
- You can also change any image at any time from the **Live Editor** (log in, click
  Edit, then click an image to replace it) — that override is saved in the browser
  and takes priority over the file. To go back to the file version, clear it in the
  editor.
- Want different filenames or more than 3 gallery photos per cottage? That's a quick
  change in `index.html` — just ask.
