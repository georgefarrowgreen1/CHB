# Cottage Holidays Blakeney

The website and back office for [Cottage Holidays Blakeney](https://www.cottageholidaysblakeney.co.uk)
— three family-run holiday cottages in Blakeney on the North Norfolk coast.

A single-operator holiday-let PWA: vanilla JavaScript and flat PHP, no
framework, no build step. Booking enquiries, Square payments, iCal channel
sync, guest messaging and a phone-first owner back office.

This repository is public so its CI can run, **not** open source — see
[LICENSE](LICENSE). Please don't reuse the code or content.
